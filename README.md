Mobitouch.net replica

This is a simplified static replica of https://mobitouch.net/ built with plain HTML, CSS and JS.

Files:
- index.html — main page
- css/styles.css — styles
- js/main.js — small interactions
- assets/ — placeholder images and logos

How to run:
1. Open `index.html` in your browser (double-click or use `Open With`).

Notes:
- Images are placeholders. Replace `assets/phone.png` and `assets/car.png` with real images for a more accurate look.
- This is a visual replica only — no backend or CMS.
